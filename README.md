# Sombras Proyecto

## Ejecutar Programa
1. Crear Entorno Virtual:
```python3
    python -m venv $PWD
```
2. Ejecutar Entorno Virtual:
- Linux, MacOS (Bash):
```sh
    source ./bin/activate
```
- Windows (PowerShell):
```powershell
    source ./bin/Activate.ps1
```
3. Instalar Librerias:
```
    python -m pip install -r requirements.txt
```
4. Ejecutar Servidor:
```python3
    python ./src/server.py
```
5. Visualizar la Pagina en `http://127.0.0.1:5000`
